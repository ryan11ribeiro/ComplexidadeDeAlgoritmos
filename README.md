# Complexidade de Algoritmos – Estruturas de Dados 2

Este repositório contém o código utilizado no vídeo sobre **Complexidade de Algoritmos**, apresentado por Ryan Ribeiro.

## Conteúdo abordado
- O(1) – acesso constante  
- O(n) – busca linear  
- O(log n) – busca binária  
- O(n²) – bubble sort
